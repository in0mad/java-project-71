## About
A difference finder this is a program that determines the differnce between two of data structures. The program takes two files and show the difference on the screen.

## Setup

```bash
make build
```

## Clean

```bash
make clean
```

## Start

```bash
make run-dist
```

## Run checkstyle

```bash
make lint
```

## Make JaCoCo Report

```bash
make report
```

## Launch the Test

```bash
make test
```
